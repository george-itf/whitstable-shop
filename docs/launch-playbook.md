# Launch Playbook

A detailed week-by-week guide to launching whitstable.shop successfully.

---

## Pre-Launch: Week -2 to 0

### Day -14: Soft Launch

**Objective**: Get early feedback from friendly testers

**Tasks**:
- [ ] Deploy to production URL
- [ ] Invite 5-10 friendly shop owners to test
- [ ] Share with friends/family in Whitstable
- [ ] Create feedback collection form
- [ ] Monitor for bugs and issues

**Success Criteria**:
- 5+ people have tested
- No critical bugs reported
- Positive initial feedback

---

### Day -10: Bug Fixes & Refinements

**Objective**: Address soft launch feedback

**Tasks**:
- [ ] Fix any reported bugs
- [ ] Implement quick-win suggestions
- [ ] Refine copy based on feedback
- [ ] Test all flows end-to-end
- [ ] Verify mobile experience

---

### Day -7: Prepare Launch Materials

**Objective**: Everything ready for go-live

**Physical Materials**:
- [ ] Print 100 shop letters (A5, folded)
- [ ] Print 100 business cards with QR code
- [ ] Order window stickers (if budget allows)
- [ ] Prepare envelope materials

**Digital Materials**:
- [ ] Create social media graphics (5 variations)
- [ ] Write launch email for newsletter
- [ ] Prepare Facebook/Instagram posts
- [ ] Draft press release
- [ ] Create Google Docs with all copy

**Email Setup**:
- [ ] Configure email newsletter (Buttondown)
- [ ] Import any existing subscribers
- [ ] Create welcome email sequence
- [ ] Test email deliverability

---

### Day -5: Content Preparation

**Objective**: Have launch day content ready

**Tasks**:
- [ ] Write 5 launch day social posts
- [ ] Prepare shop spotlight posts for week 1
- [ ] Create story templates
- [ ] Draft local group posts (follow each group's rules!)
- [ ] Prepare responses to common questions

---

### Day -3: Shop Outreach

**Objective**: Personal introduction to shops

**Route Planning**:
- Map efficient walking route through Harbour Street, High Street, Oxford Street
- Identify 20 priority shops to visit personally
- Prepare talking points

**Personal Visits**:
- [ ] Visit 10-20 shops in person
- [ ] Introduce yourself briefly (30 seconds max)
- [ ] Hand over envelope with letter + card
- [ ] Note which shops seem interested
- [ ] Don't be pushy - plant seeds

**Talking Script**:
> "Hi, I'm [Name]. I've built a new website called whitstable.shop to help visitors discover independent shops like yours. I've included your shop, and wanted to drop off some info about how you can claim your free listing. No pressure - just wanted to say hello!"

**Letter Delivery**:
- [ ] Hand-deliver envelopes to remaining shops
- [ ] Leave with staff if owner not available
- [ ] Note any closed shops for follow-up

---

### Day -1: Final Checks

**Technical**:
- [ ] All pages loading correctly
- [ ] SSL certificate valid
- [ ] Analytics tracking working
- [ ] Email signup working
- [ ] Map loading properly
- [ ] Mobile tested on iOS and Android

**Content**:
- [ ] All shop listings accurate
- [ ] No placeholder text remaining
- [ ] Images loading correctly
- [ ] Links working

**Team**:
- [ ] Everyone knows their launch day role
- [ ] Support email monitored
- [ ] Social media access confirmed

**Teaser Post**:
- [ ] Post "launching tomorrow" teaser on social
- [ ] Send teaser email to list

---

## Launch Day

### Morning (8-9am)

**Go Live Announcement**:

1. **Instagram Post**
   > "We're live! whitstable.shop is now open - your new guide to independent Whitstable. Discover 80+ shops, read reviews, find what's on, and more.
   >
   > Link in bio"

2. **Instagram Stories**
   - Swipeable intro to features
   - Behind-the-scenes of building it
   - Thank you to early supporters

3. **Facebook Page Post**
   - Similar content to Instagram
   - Include direct link

4. **Email Newsletter**
   > Subject: whitstable.shop is live!
   >
   > After months of work, whitstable.shop is finally open...

5. **WhatsApp to Interested Shops**
   > "Hi [Name]! Just wanted to let you know whitstable.shop is now live. Your shop is featured at [link]. Let me know if you have any questions!"

---

### Mid-Morning (10-11am)

**Local Facebook Groups**:

Post in (check each group's rules first):
- Whitstable Community Group
- What's On in Whitstable
- Whitstable Mums/Parents groups (if appropriate)
- Whitstable Recommendations

**Sample Post**:
> "Hi everyone! I've just launched whitstable.shop - a free directory of independent shops in Whitstable. You can browse shops, read reviews, check what's on, and even find tide times and parking info. Would love to hear what you think! [link]
>
> (Mods - happy to remove if not appropriate, just wanted to share something hopefully useful for the community!)"

---

### Afternoon (2-4pm)

**Monitor & Respond**:
- [ ] Check for comments on all posts
- [ ] Respond to every comment same day
- [ ] Monitor email for questions
- [ ] Check analytics for any issues
- [ ] Note any bug reports

**Engagement**:
- [ ] Thank people who share
- [ ] Answer questions helpfully
- [ ] Don't be defensive about criticism

---

### Evening (6-8pm)

**Thank You Post**:
- [ ] Post thank you on Instagram stories
- [ ] Share any early screenshots (with permission)
- [ ] Highlight first reviews or saves

**Team Debrief**:
- [ ] What went well?
- [ ] Any issues to fix tomorrow?
- [ ] Capture initial metrics

---

## Post-Launch: Week 1

### Daily Tasks

- [ ] Check and respond to all comments/messages
- [ ] Monitor analytics
- [ ] Fix any reported issues
- [ ] Post daily social content

### Day 2: Shop Spotlights Begin

Start daily shop spotlight posts:
- One shop per day
- Good photo + brief description
- Tag the shop if they're on social
- Mix categories (food, retail, services)

### Day 3-4: Press Outreach

Send press release to:
- [ ] Whitstable Gazette (news@whitstablegazette.co.uk)
- [ ] Kent Online
- [ ] Canterbury Times
- [ ] KentLive

Follow up with phone call if no response in 48 hours.

### Day 5: Shop Follow-Up

- [ ] Email shops who received letters
- [ ] Visit any who expressed strong interest
- [ ] Offer help claiming listings
- [ ] Note any concerns or questions

### Day 7: Week 1 Review

**Metrics to Check**:
- Total visitors
- New user signups
- Reviews submitted
- Shops claimed
- Email subscribers

**Content Review**:
- Which posts performed best?
- What questions are people asking?
- Any gaps in content?

## Contingency Plans

### If Traffic is Low

- Increase social posting frequency
- Run targeted Facebook ad (Â£20-50 budget)
- More local group posting
- Personal outreach to Whitstable contacts

### If Bugs Emerge

- Prioritise user-facing issues
- Communicate transparently
- Fix fast, apologise sincerely
- Document for future prevention

### If Negative Feedback

- Thank for feedback genuinely
- Don't be defensive
- Fix legitimate issues quickly
- Ignore trolls (don't engage)

### If Shops Unhappy

- Reach out personally
- Understand concerns
- Make reasonable accommodations
- Explain the value proposition
