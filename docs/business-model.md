# Business Model

A phased approach to building a sustainable, profitable platform while keeping core features free for visitors.

---

## Vision

whitstable.shop becomes the essential digital infrastructure for Whitstable's independent high street - trusted by visitors, valued by shop owners, and sustainable for the long term.

---

## Phase 1: Free (Launch - Month 6)

### Strategy

Build trust, adoption, and content. Prove the concept before asking anyone to pay.

### All Features Free

- Shop listings
- Reviews and ratings
- Map and search
- Events calendar
- Local information
- User accounts and saves
- Basic shop owner profiles

### Goals

| Metric | Target |
|--------|--------|
| Shops listed | 80+ |
| Monthly unique visitors | 500+ |
| Total reviews | 50+ |
| Claimed shops | 10+ |
| Email subscribers | 200+ |

### Focus

1. Complete shop database
2. Generate initial reviews
3. Build brand awareness
4. Establish trust with shop owners
5. Gather feedback for premium features

### Revenue

**£0** - This is an investment phase.

---

## Phase 2: Premium Features (Month 6+)

### Shop Owner Premium (£5-15/month)

Value proposition: Visibility, insights, and tools to attract more customers.

#### Tier 1: Enhanced Listing (£5/month)

- **Featured badge** on listing
- **Photo gallery** (10 photos vs. 3 free)
- **Special offers section** on profile
- **Opening hours exceptions** (bank holidays, etc.)
- **Priority support** via email

#### Tier 2: Analytics + Engagement (£10/month)

Everything in Tier 1, plus:

- **Analytics dashboard**
  - Profile views
  - Map marker clicks
  - Direction/call clicks
  - Save count
  - Search impressions
- **Review response** - Reply publicly to reviews
- **Weekly email report** with stats
- **Competitor insights** (anonymised category trends)

#### Tier 3: Featured Placement (£15/month)

Everything in Tier 2, plus:

- **Featured in category** - Top of category listings
- **Homepage spotlight** - Rotating featured position
- **Social media shoutout** - Monthly feature on our Instagram
- **Premium badge** - "Featured Shop" designation

### Pricing Rationale

- £5-15/month is affordable for small businesses
- Less than a Google Ads click in competitive markets
- Comparable to other local directory premiums
- Can be cancelled anytime - no contracts

### Visitor Features (Free Forever)

Core functionality remains free:

- Browse all shops
- Read all reviews
- Use map and search
- View events
- Create account
- Save shops
- Write reviews

**Philosophy**: Visitors should never pay. They provide the value (reviews, traffic) that makes premium worthwhile for shops.

## Phase 3: Partnerships (Year 2+)

### Tourism Partnerships

**Canterbury City Council / Visit Kent**

- Annual partnership fee for integration
- Cross-promotion of tourism content
- Data sharing (anonymised visitor insights)

### Media Partnerships

**Local Publications (Kent Online, Whitstable Gazette)**

- Sponsored content partnerships
- Event promotion

**Potential value**: £500-1,500/year

### Event Partnerships

**Festival/Event Organisers**

- Premium event listings
- Dedicated event microsites
- Ticket integration

**Potential value**: £100-500 per major event

| **Total** | **£10-115** | |

### Variable Costs

- Payment processing: 2-3% of revenue
- Support time: Scale with customers
- Marketing: As budget allows

### Break-Even

With £50-100/month in costs, need:
- 5-10 premium subscribers at £10/month
- Achievable by Month 9-12

## Competitive Positioning

### Why Shop Owners Pay Us

1. **Local focus**: Not competing with chains/global brands
2. **Quality audience**: Visitors actively seeking local shops
3. **Affordable**: Fraction of Google/Facebook ad costs
4. **Measurable**: Clear analytics on what they get
5. **Community**: Part of Whitstable's digital presence

### Why Not Google/Facebook Ads?

| Factor | Google Ads | whitstable.shop |
|--------|------------|-----------------|
| Cost | £1-5 per click | £10/month flat |
| Targeting | Broad | Hyper-local |
| Intent | Mixed | High (seeking local) |
| Competition | High | Low |
| Brand alignment | Generic | Local/community |

## Risk Mitigation

### Risk: Low Shop Owner Conversion

**Mitigation**:
- Start with lower prices
- Offer longer free trials
- Focus on demonstrable value (analytics)
- Build case studies from early adopters

### Risk: Low Visitor Traffic

**Mitigation**:
- SEO investment
- Tourism partnerships
- Shop owner promotion (referral program)
- Local PR and media

### Risk: Competition

**Mitigation**:
- First-mover advantage in Whitstable
- Deep local knowledge and relationships
- Community trust built over time
- Features tailored to local needs

## Success Metrics

### Health Metrics (Track Weekly)

- Unique visitors
- Shops claimed
- Reviews submitted
- Premium conversion rate

### Growth Metrics (Track Monthly)

- MRR (Monthly Recurring Revenue)
- Churn rate
- CAC (Customer Acquisition Cost)
- LTV (Lifetime Value)

### Target Ratios

- LTV:CAC > 3:1
- Churn < 5% monthly
- NPS > 40

## Decision Points

### Month 6: Launch Premium?

Evaluate:
- Do we have 50+ shops claimed?
- Do we have consistent traffic (500+ monthly)?
- Have shop owners expressed interest in premium?

If yes → Launch Tier 1 only
If no → Extend free period, focus on growth

### Month 12: Expand Premium?

Evaluate:
- Premium conversion > 15%?
- Churn < 10%?
- Positive feedback?

If yes → Launch Tier 2 and 3
If no → Iterate on Tier 1, gather feedback

### Month 18: Partnerships?

Evaluate:
- Established brand in Whitstable?
- Tourism interest?
- Resource to manage partnerships?

If yes → Pursue council/tourism partnerships
If no → Focus on core product

---

*Review this model quarterly and adjust based on actual performance.*
